# Swiss Leadership Consultancy

Premium static landing page for Swiss Leadership Consultancy, ready for GitHub Pages and other static hosting platforms.

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Local Preview

Run a simple static server from the project root:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deploy

This site is deployment-ready for:

- GitHub Pages
- Netlify
- Vercel static hosting
- Traditional hosting via direct file upload

### GitHub Pages

The site is configured as a root-level static site, so GitHub Pages only needs to publish from the repository root or the built branch that contains these files.

Required files:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Assets

The `assets/` folder contains the media used by the landing page:

- `logo.png`
- `team.jpg`
- `hero-video.mp4`
